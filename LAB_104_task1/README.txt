# LAB 104 - Task 1

## Project Description
This repository contains files and documentation for Lab 104 Task 1. The project is part of the coursework at the university.

## Overview
The project consists of a fully implemented Sequence class with 2 additional functions in split.hpp whose purpose is to divide the list.

## Structure
- .vscode folder — project build configuratio
- sequence.hpp — declaration of the Sequence class
- split.hpp — additional functions for splitting the list
- main.cpp — example program / usage demonstration with unit tests
- README.txt — project description (this file)
- sequence.exe - executable file

## Setup
works fine on VScode after command "code ." is applied, if not the compiler might be a problem. If not try commands below:
1. g++ main.cpp -o sequence.exe
2. ./sequence.exe
the unit tests should be executed
